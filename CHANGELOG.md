# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [0.1.0] - 2026-03-11

### Added

- Windows desktop application built with PySide6
- Server CRUD with SSH connectivity testing
- Script CRUD for reusable shell scripts
- Manual command execution across one or multiple servers
- Script execution across one or multiple servers
- Background execution with per-host and aggregate console views
- Sudo and PTY support for privileged commands
- AI analysis with OpenAI, Ollama, and OpenAI-compatible providers
- Suggested actions and structured fix plans
- Approve / reject / execute flow for AI-suggested steps
- Incident Mode Phase 1 with safe diagnostic orchestration
- Run history UI with replayable output and AI linkage
- About dialog, System Info dialog, runtime logging, and Windows packaging support
